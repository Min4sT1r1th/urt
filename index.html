URT Project Documentation

This project was dormant since the last release of 3.1b (around 1990 or so) and is now found pretty much only found on historical Unix archive sites (upstream home is just HTML-ified man pages). This release is mainly build and compiler fixes, along with a new shared library for RLE consumers; the old pbmplus dependency is not curently supported but all the other tools build and this package satisfies the netpbm dependency for RLE support. Some older constructs have been updated and project makefiles fixed to both build manually and via ebuild package with new docs and metrics targets (see latest release output below). Other enhancements and/or extensions may also happen; stay tuned...

  • User Guides - Upstream user guide (may not be current).
  • Software API docs - Software API documentation with design diagrams, etc.
  • Complexity Metrics - Source code metrics for a source code metrics tool... ;)
