10  * Usage:
11  * rlescale [-c] [-l] [-n nsteps] [xsize] [ysize]
12  *
13  * Generates a xsize x ysize gray-scale RLE file (by default 480x512).
14  * Has color squares along the bottom, with nsteps (default 16) gray
